# Sneha-Date 💕

**A Special Interactive Website For My Maharani** ❤️

## 📁 Project Structure

```
Sneha-Date/
├── index.html      ← Main page (7 interactive pages)
├── style.css       ← Beautiful gradient & animations
├── script.js       ← Interactive features & page navigation
└── README.md       ← This file
```

## ✨ Features

✅ **7 Interactive Pages** - Beautiful story progression  
✅ **Smooth Animations** - Fade effects, bouncing emojis, floating effects  
✅ **Gradient Background** - Purple to pink gradient  
✅ **Glassmorphism Design** - Modern frosted glass effect  
✅ **Running "No" Button** - Playful interaction on page 6  
✅ **Click Hearts** - Hearts appear when clicking anywhere  
✅ **Keyboard Support** - Press Enter to go to next page  
✅ **Responsive Design** - Works on mobile and desktop  

## 🚀 How to Use

1. **Open the website** - Simply open `index.html` in your browser
2. **Click "Continue"** - Navigate through each page
3. **On the final page** - Click "YES!" to confess your love 💕
4. **Try the "No" button** - Watch it run away!
5. **Click anywhere** - Hearts will float up
6. **Press Enter** - Quick navigation between pages

## 🎨 Customization

### Change Colors
Edit `style.css` - Look for:
```css
background:linear-gradient(
    180deg,
    #12002f,   /* Dark purple */
    #2d0b52,   /* Medium purple */
    #ff5c8a    /* Pink */
);
```

### Change Text
Edit `index.html` - Modify the content inside each page div
